dllexport, dllimport

Visual Studio 2012
Другие версии
  • Visual Studio 2010

Только для систем Microsoft

dllexport и dllimport атрибуты хранени-класса расширения Майкрософт-специфического на языках c и C++. Они позволяют экспортировать и импортировать функции, данные и объекты и из библиотек DLL.

Копировать 
__declspec( dllimport ) declarator
__declspec( dllexport ) declarator
Заметки

Эти атрибуты явно указывают интерфейс DLL к своему клиенту, который может быть исполняемым файлом или другую библиотеку DLL. Объявление функции как dllexport исключает необходимость для файла определения модуля (def), по крайней мере, по отношению к спецификации экспортированных функций. Обратите внимание, что dllexport заменяет __export ключевое слово.

Если класс помечен как спецификация (dllexport) все специализации шаблонов класса в иерархии классов явно помечены как спецификация (dllexport). Это означает, что шаблоны явно создается и его члены.

dllexport функции c++ предоставляет функцию с mangling имени C++. Если не пожелан mangling имени C++, либо следует использовать def-файл (ЭКСПОРТЫ ключевое слово) или как объявляется функция extern "c".

Объявление dllexport и dllimport использование расширенный синтаксис атрибутов и __declspec ключевое слово.

Пример
Копировать 
// Example of the dllimport and dllexport class attributes
__declspec( dllimport ) int i;
__declspec( dllexport ) void func();

Также сделать код более четким, можно использовать определения макроса:

Копировать 
#define DllImport   __declspec( dllimport )
#define DllExport   __declspec( dllexport )

DllExport void func();
DllExport int i = 10;
DllImport int j;
DllExport int n;
